# Teste JSF

Um pequeno teste de desenvolvimento de um Web App em JSF.

# Requisitos

1) JBoss EAP 7.0.
2) Banco de dados PostgreSQL (configurável no persistance.xml e mediante fornecimento de driver).

## Instruções para ambiente Eclipse

1) Importe o projeto como um projeto Maven.
2) Configure no persistance.xml parâmetros da sua conexão com o banco.
3) Nas propriedades do projeto, em "Targeted Runtimes", selecione o JBoss alvo.
4) Selecione Debug As/Run As ou Export WAR File, e efetue o deploy do WAR no Jboss.
